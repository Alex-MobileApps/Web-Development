About

Swift is a programming language developed by Apple Inc. for use on iOS, macOS, watchOS, tvOS and more. It was intended to support many concepts from Objective-C, but in a safer way. The compiler is optimised for performance and the language is optimised for development.

Swift avoids many common programming errors:

  • Variables are always initialised before use.
  • Array indices are checked for out-of-bounds errors.
  • Integers are checked for overflow.
  • Handling of nil values with Optionals.
  • Memory is automatically managed

Swift was released in 2014. It has been open source late 2015. This documentation has been updated to match the release of Swift 5.1.

Constants and Variables

A constant or variable is used to associate a name with a value of a certain type. They must be declared before they are used. Swift is quite intellient, it can often infer the type of value you are using without needed to explicitly declare it yourself.

Constants are immutable and are declared with the let keyword. For example:

let myName = "Alex"

Variables are mutuable and are declared with the var keyword. For example:

var remainingTime = 9

As you can see above, Swift was able to determine that myName is a string and remainingTime is an integer without having to state it. The two values could also be annotated like this:

let myName: String = "Alex"
var remainingTime: Int = 9

Optionals are a powerful feature in Swift. An optional is a variable than can hold the value nil. Here is an example of the creation of an optional variable:

var firstCaller: String?

If you know an optional value is going to be set when it is called, it can be implicitly unwrapped by using an exclamation mark:

var winningNumber: Int!
Comments and Printing

Comments are used to include nonexecutable text within your code. They are ignored by the Swift compiler when executing. A single line comment begins with two forward slashes (//...) and a multiline comment is enclosed in a forward slash and an asterisk (/*...*/).

// This is a single line comment.
/* This is a
multiline comment. */

The value of a constant or variable can be printed with the print(_:separator:terminator:) function. In our earlier example, we could print our name to the console:

print(myName)
//Prints "Alex"

Swift uses String Interpolation to allow constants and variables to be included in a longer string. This can be seen in the following print statement:

print("Attention \(myName), you have \(remainingTime) minutes remaining!")
//Prints "Attention Alex, you have 9 minutes remaining!"
Conditional Statements

It is often important when coding to use different code depending on certain conditions. For example, you might want to display a congratulations message when something is successful, or the opposite otherwise.

The if statement is used to execute code if a condition is true. For example:

if remainingTime == 0 {
    print("You are out of time!")
}
//Nothing is printed as remainingTime does not equal 0

The else statement can be paired with an if statement to offer an alternative if the if statement returns false. Using our example above:

if remainingTime == 0 {
    print("You are out of time!")
} else {
    print("You still have time!")
}
//Prints "You still have time!"

Another important conditional statement available in Swift is the switch statement. A switch statement is like an if else statement but it allows for multiple cases without having to re-evaluate the condition each time. This statement must be exhaustive, that is that every possible case must be included in the switch statement. If you can't provide a case for every value, the default keyword must be used for all other possible values.

let myPasta: String = "Carbonara"
switch myPasta {
case "Marinara":
    print("This is a seafood dish")
case "Carbonara":
    print("This is a cheese and egg dish")
case "Bolognese":
    print("This is a meat dish")
default:
    print("This is another pasta sauce")
}
//Prints "This is a cheese and egg dish"
Functions

A function is a group of code that is used to perform a certain task. Every function must have a name and begin with the func keyword. When a function is called, the code contained within is executed. For example, we might create the following function:

func getDetails() {
    print("Name: \(myName), Favourite Pasta: \(myPasta)")
}

We can now run this code by calling the function:

getDetails()
//Prints "Name: Alex, Favourite Pasta: Carbonara"

At the moment our function is pretty rigid. What if we want to use values other than our own name and our favourite pasta? We can do this by adding input values to the function just like this:

func getDetails(name: String, pasta: String) {
    print("Name: \(name), Favourite Pasta: \(pasta)")
}

If we run our new code with the strings "Sam" and "Bolognese" we would get:

getDetails(name: "Sam", pasta: "Bolognese")
//Prints "Name: Sam, Favourite Pasta: Bolognese"

Functions can also have return types. A variable or constant can be set using the value returned from a function. Here is an example of a function with a string return type:

func isItBolognese(name: String, pasta: String) -> String {
    if pasta == "Bolognese" {
        return "Yes, it is Bolognese"
    } else {
        return "No, it is another sauce"
    }
}

And when called, here is our output:

isItBolognese(name: myName, pasta: myPasta)
//Returns "No, it is another sauce"
